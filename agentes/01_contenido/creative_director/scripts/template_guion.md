# Template de guion — Brief de producción DV

Este template es la versión operativa de la Parte 5 del sistema de video ads. Es el documento que llega al set: lo lee Bauti CB para producir, lo lee el editor para cortar, y el cliente lo ve antes de grabar para saber exactamente qué va a decir y qué se va a filmar.

Todas las secciones son obligatorias salvo las marcadas como (opcional). Si una sección no aplica, escribís "no aplica" y por qué. Nunca la dejes en blanco.

---

## 7. Validación de ratio valor:tiempo

Dividir el guion mentalmente en bloques de 5-8 segundos. Para cada bloque escribir en una línea: ¿qué valor entrega este bloque (educa, entretiene, inspira)? Si algún bloque no entrega valor, marcarlo y reescribir o cortar antes de aprobar el guion.

- Bloque 0-8s: [valor que entrega]
- Bloque 8-16s: [valor]
- Bloque 16-24s: [valor]
- [seguir hasta el final del video]

Resultado del check: [ratio alto / medio / bajo]. Si es medio o bajo, no se aprueba para grabación.

## 11. Checklist de producción en set

**Antes de grabar**
- [ ] Guion impreso o en teleprompter
- [ ] Formato 9:16 configurado en cámara
- [ ] Iluminación: sin sombras en la cara
- [ ] Fondo coherente con el posicionamiento
- [ ] Ropa sólida (evitar rayas/logos)
- [ ] Subtítulos planificados para edición
- [ ] Batería y almacenamiento chequeados

**Durante y después de grabar**
- [ ] Mínimo 3 tomas del hook (las 3 variantes A/B/C grabadas)
- [ ] B-roll de apoyo para cada bloque clave
- [ ] Toma del CTA con distintas intensidades
- [ ] Video en horizontal extra para posible adaptación
- [ ] Takes sin guion para naturalidad
- [ ] Carpeta organizada por formato/sección
- [ ] Backup inmediato del material
